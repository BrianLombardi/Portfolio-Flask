Projetos

Projeto 1: Aplicativo de Lista de Tarefas

Desenvolvi um aplicativo simples de lista de tarefas utilizando HTML, CSS e JavaScript. O projeto permite aos usuários adicionar, editar e remover tarefas, e visualizá-las em uma lista organizada. Este foi um ótimo exercício para aprimorar minhas habilidades em desenvolvimento front-end e gerenciamento de estado.

Projeto 2: Site Pessoal

Criei um site pessoal para exibir minhas habilidades e projetos. Utilizei HTML e CSS para o design e a estrutura do site, com um layout responsivo que se adapta a diferentes tamanhos de tela. O site inclui seções sobre mim, meus projetos e informações de contato.

Projeto 3: Sistema de Registro de Usuários

Desenvolvi um sistema básico de registro de usuários com Python e Flask. Este projeto inclui funcionalidades para cadastro e autenticação de usuários, e um painel de administração simples para gerenciar as informações dos usuários. Foi uma excelente oportunidade para aprender mais sobre desenvolvimento back-end e bancos de dados.

Projeto 4: API de Clima

Criei uma API simples que fornece informações sobre o clima usando uma API pública de clima. O projeto foi feito com Python e Flask, e permite que os usuários consultem as condições meteorológicas atuais para diferentes cidades. Este projeto me ajudou a entender melhor a integração de APIs e o tratamento de dados JSON.
